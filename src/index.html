Titanfall 2 Randomizer

Hints

Use the menu to open the level you're satuck on. All the levels are hidden by default to avoid spoilers, however you can use the Show All button to display all the levels at once.

About

The Titanfall 2 Randomizer takes the main campaign gameplay and tuns it on its head. All the guns are random, all the abilities are random, the Titan classes available are random, and the helmet collectables are random. Each level has 15-30 random positions
